## Обзор

`remote-sdr-radio` — проект программно-аппаратной системы радиоприёма, обеспечивающей:

- приём радиосигнала с использованием `RTL-SDR`;
- программную обработку сигнала в `GNU Radio`;
- формирование аудиопотока;
- удалённое управление системой через браузер.

Система может использоваться как в учебных целях, так и в качестве основы для последующего развития в сторону спектрального анализа и радиомониторинга.

## Функциональность

- удалённое управление через веб-интерфейс;
- изменение частоты приёма;
- управление параметрами радиоприёма;
- формирование аудиопотока в режиме реального времени;
- воспроизведение сигнала в браузере;
- удалённый доступ к Raspberry Pi по SSH;
- возможность использования в локальной сети и через туннелирование.

## Аппаратная конфигурация

| Компонент | Назначение |
|---|---|
| Raspberry Pi 5 | Вычислительная платформа |
| RTL-SDR Terratec | Приём радиосигнала |
| microSD | Носитель операционной системы |
| Ethernet / Wi-Fi | Сетевое подключение |
| Антенна | Приём радиосигнала |

## Архитектура

Логика системы включает следующие этапы:

1. `RTL-SDR` принимает радиосигнал.
2. `GNU Radio` выполняет фильтрацию и демодуляцию.
3. Аудиоданные передаются в `FFmpeg`.
4. `FFmpeg` формирует HTTP-аудиопоток.
5. `Flask` предоставляет веб-интерфейс управления.
6. Пользователь получает доступ к управлению и аудиопотоку через браузер.
