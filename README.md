# Remote Sdr Radio
### Система удалённого радиоприёма на базе SDR-приёмника и Raspberry Pi

![Python](https://img.shields.io/badge/Python-3.x-blue)
![GNU Radio](https://img.shields.io/badge/GNU%20Radio-SDR-orange)
![Flask](https://img.shields.io/badge/Flask-Web%20Interface-black)

Проект демонстрирует реализацию удалённого радиоприёмника на базе `RTL-SDR` и `Raspberry Pi` с обработкой сигнала в `GNU Radio`, передачей аудиопотока через `FFmpeg` и управлением через веб-интерфейс на `Flask`.

## Обзор

Проект демонстрирует реализацию удалённого радиоприёмника: приём сигнала с RTL‑SDR, обработка в GNU Radio, кодирование аудио через FFmpeg и управление через веб‑интерфейс на Flask.

---

## Возможности

* изменение частоты приёма;
* управление параметрами приёмника;
* потоковая передача аудио (HTTP);
* воспроизведение в браузере;
* удалённый доступ по SSH;
* работа в локальной сети и через туннелирование.

---

## Аппаратная часть

| Компонент          | Назначение          |
| ------------------ | ------------------- |
| Raspberry Pi 5     | Вычислительный узел |
| RTL‑SDR (RTL2832U) | Приём радиосигнала  |
| microSD            | Хранение системы    |
| Антенна            | Приём сигнала       |

---

## Архитектура

1. RTL‑SDR принимает радиосигнал
2. GNU Radio выполняет фильтрацию и демодуляцию
3. FFmpeg кодирует аудио и публикует поток
4. Flask отдаёт веб‑интерфейс и API
5. Браузер воспроизводит поток и отправляет команды

## Установка

```bash
sudo apt update
sudo apt upgrade -y
sudo apt install python3 python3-pip ffmpeg gnuradio
pip3 install -r requirements.txt
```

Проверка SDR:

```bash
rtl_test -t
```

---

## Быстрый запуск

```bash
chmod +x start_remote_radio.sh
./start_remote_radio.sh
```

Остановка:

```bash
chmod +x kill_all_radio.sh
./kill_all_radio.sh
```
